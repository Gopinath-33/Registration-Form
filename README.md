# Registration Form

A simple, clean user registration form built with HTML, CSS, and JavaScript. Deployed live on GitHub Pages.

🔗 **Live Demo:** [gopinath-33.github.io/Registration-Form](https://gopinath-33.github.io/Registration-Form/)

## 📋 Overview

This project is a front-end registration form that collects basic user details — Name, Email, and Password — through a clean, easy-to-use interface. It's a foundational web development project focused on form structure, input validation, and responsive UI design.

## ✨ Features

- Simple and intuitive registration form UI
- Fields for Name, Email, and Password
- Client-side form validation
- Responsive design for mobile and desktop
- Clean, minimal styling for better user experience

## 🛠️ Tech Stack

- **HTML5** – Form structure and markup
- **CSS3** – Styling and layout
- **JavaScript** – Form validation and interactivity

## 📁 Project Structure

```
Registration-Form/
├── index.html      # Main HTML structure
├── style.css       # Styling for the form
├── script.js       # Form validation logic
└── README.md       # Project documentation
```

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
   ```
   git clone https://github.com/Gopinath-33/Registration-Form.git
   ```
2. Navigate to the project folder
   ```
   cd Registration-Form
   ```
3. Open `index.html` in your browser
